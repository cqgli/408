# 一元积分学的概念和性质

1. 不定积分
   1. 连续函数必有原函数
   2. 含有非振荡间断点的间断点的函数区间内没有原函数
   3. fx可导，且fx在x0极限存在，则导数在x0连续
   4. 导数存在，则存在导数零点定理
2. 定积分
   1. 充分条件
      1. fx在有限区间连续，则定积分存在
      2. fx在有限区间单调，则定积分存在
      3. fx在有限区间有界且只有有限个非无穷间断点的，则定积分存在
      4. fx在有限区间有有限个第一类间断点，则定积分存在
   2. 必要条件
      1. 定积分存在，fx在区间有界
   3. 性质
      1. 求区间
      2. 线性性质
      3. 可加性
      4. 保号性
      5. 估值定理
      6. 中值定理
3. 变限积分
   1. fx可积在区间，则原函数连续
   2. fx连续在区间，则原函数可导且导数为fx
   3. fx存在跳跃间断点，则原函数在间断点不可导
   4. fx存在可去间断点，则原函数在间断点可导
4. 反常积分
