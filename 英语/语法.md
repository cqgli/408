# 简单句

## 简单句的核心构成

1. 主谓 = 主语+不及物动词（vi）
2. 主谓宾 = 主语+及物动词（vt）+宾语
3. 主谓双宾 = 主语+及物动词（vt）+宾语（一般为人）+宾语（物）
   - ​	= 主语+及物动词（vt）+宾语（物）+介词（to或for) +宾语（人）
4. 主谓宾宾补 = 主语+及物动词（vt)+宾语+宾语补足语
   - 一般宾语+是+宾语补足语可以构成句子
5. 主系表 = 主语+系动词+表语
   - 系动词分类
     - ​	be动词
     - get ，become， turn，go，grow 表示**变得，成为**时
     - look，sound，taste，smell，feel 感官动词
     - seem，appear 表示**似乎**时
     - keep，remain表示**保持**时
     - prove 表示**证据，显示出**时

## 简单句的核心变化

### 谓语动词的变化

#### 谓语动词的时态

1. 常考时态：一般现在时，一般过去时，一般将来时，过去将来时，过去进行时，过去完成时，现在进行时，现在完成时

2. 一般过去时：did表示完全发生在过去的事情

3. 一般现在时：do/does表示现在的状态，可以是经常发生，习惯性发生的事。

   - ​	第三人称单数的主语一般有he，she，it，还有不定代词someone，Everyone，something，nobody等，还有不可数名词，可数名词单数

4. 一般将来时：will do，be going to do，（当第一人称时可以用  shall do表示）表示现在的将来

5. 过去将来时：would do，was/were going to do 表示发生在过去的将来，通常与上下文的过去时态进行搭配

   They **gave** justices permanent position so they **would be** free to upset those in power。

6. 过去进行时：was/were doing 强调过去某个时间段正在进行或持续进行的事情

7. 现在进行时：is，are doing 强调现在正在进行的事情

8. 过去完成时：had done 表示过去的之前（表示一个时间段）

9. 现在完成时：have、has+done表示现在的之前

   表示从过去到现在的时间段

   当表示全部完成的时候可以与一般过去时互换使用

   当表示部分完成时不可以互换使用

- ​		在have + 介词+done,介词为already已经，just 刚刚，ever 曾经，never 从不，also 也，nearly 几乎，long长期

#### 谓语动词的情态

1. 谓语动词表示**情态**
   - must 必须
   - can，could 能够，可以
   - will，would 将来，愿意
   - may，might 可以，可能
   - should 应该
2. 谓语动词表示**推测**
   - must  最可能，can't could't 最不可能
   - 情态动词+be
   - 表示过去 情态动词+have、has +done
   - 表示进行 情态动词+be doing
   - 常用表情态动词的词组 have' to 必须，ought  to 应该

#### 谓语动词的语态

1. 主动语态：主语是谓语动词的动作发出者

2. 被动语态：主语是谓语动词的动作承受者

   - 谓语动词构成 ：be +done
   - 谓语动词确定方法
     - 被动的动作（改为done形式）
     - 动作的时间（变换成be的相应时态）
     - 动作的单复数（根据主语判断be动词的单复数形式）
   - 如果使用被动语态时可以加 by+动作的发出者，可以强调被“who做”

3. 被动语态与时态的结合（改变be的形式）

   - 过去形式：was、were + done
   - 将来形式：will be +done
   - 现在形式 ：are ，is，am+ done
   - 过去将来时：would be +done
   - 过去进行时： was，were being +done
   - 现在进行时：are、am、is being +done
   - 过去完成时：had been +done
   - 现在完成时：have，has been +done

4. 被动语态与情态的结合

   - 一般形式： 情态动词+be + done

5. **被动语态在翻译中的用法**

   - 翻译时变为主动，人+....
   - 翻译时去掉被，不改变原来的意思
   - 翻译时被改为 受到了，得到了，遭受了


#### 谓语动词的否定

1. ​	分为实义动词的否定，系动词，助动词，情态动词的否定
2. 实义动词的否定：do、does、did+not + 实义动词原型（**根据时态和人称改变do的形式**)
3. be动词，系动词，情态动词，完成时态的have、has、had等变否定：直接在后面加not
   - 当谓语动词包含多个助动词或情态动词，直接在第一个动词后加not
   - is not being cheated

#### 谓语动词的强调

​	**直接在动词前加do,does,did+动词原型，根据时态和人称改变do的形式**

## 简单句的扩展

### 词性角度的扩展

#### 限定词

1. 修饰限定名词
   - 41 percent of 百分之41
   - two thirds of 三分之二，**当分子大于一时，分母为负数**
   - dozen 12个

#### 形容词和副词

1. ​	形容词修饰名词
   - 形容词在名词左右，作定语修饰
   - 名词+be动词+形容词，作表语修饰
2. 副词修饰名词外的其他
   - 修饰动词
     - 当修饰谓语动词为情态动词或助动词加实义动词时，副词在情态动词后，实义动词前
   - 修饰句子
   - 修饰形容词或其他副词
3. 形容词和副词的比较级
   - 原级比较：as+adv，adj（原型）+as+比较对象
   - 比较级：adj，adv的比较级+than+（比较的对象)----**一般隐藏对象**
   - 最高级：the + adj,adv的最高级+（介词短语表示比较的范围）

#### 介词短语

1. 表示时间的介词
   1. at+几点：表示时间点，on：表示时间上+天，in：表示时间内+年，月，季节，上下午
   2. before：在什么之前，after：在什么之后
   3. from，since：自从什么之后
   4. by，untill：截止到什么时间
   5. for：持续一段时间
   6. during：在什么期间
2. 表示地点的介词
   1. at：在什么地点 in：在什么里面
   2. on，above，over：在什么上面，under，below，beneath：在什么下面
   3. in front of:在什么之前，behind：在什么之后
   4. near：在什么附近，beside，next to，by：在什么旁边
   5. between，among：在什么之间
   6. around：在什么周围
   7. into：进入什么里面，out of：从什么出去
   8. onto：到什么上面，off：脱离
   9. across，through：穿过，past：经过
   10. along:沿着
   11. from：来自，to，towards：朝着走去
   12. beyond：超过
   13. within：在什么范围
3. 其他的介词
   1. of：a of B表示b的a
   2. about：关于
   3. with：伴随着有 without：伴随着没有
   4. for：为了，因为，对于
   5. by：通过，被做，**相差了**
   6. as:作为，像，如同
   7. like：像，例如
   8. despite，in spite of：尽管

#### 同位语，插入语

1. ​	同位语：**用来解释说明名词（一般为专有名词），通常放在名词的前后**
2. 插入语：插入语的两端通常有成对的都好，破折号，括号
   1. 主谓结构做插入语表示 **某人认为，某人说**
   2. 副词做插入语
   3. 介词短语做插入语

## 非谓语动词

### 非谓语动词作主语，宾语，表语

1. 非谓语动词doing，to do作名词作主语
   1. doing作名词时不一定只有一个词，如果是及物动词就要加一个宾语，同时还可以有其他成分修饰如形容词，介词，副词
   2. to do作名词与doing类似，他们可以在做主语时意思相近
   3. **doing一般放在句首，to do一般放在句尾，前面用it作为形式主语**
   4. 动名词做主语谓语动词用单三
   5. it is take sb some time to do sth 做某事花费了某人多长时间
2. 非谓语动词doing，to do作宾语
   1. 两种形式：动宾，介宾（**只能跟doing**）
   2. 动宾一般跟固定搭配 
      - want to do
      - keep doing
   3. **to do作宾语时还可以用it作形式宾语，但是需要在主谓宾宾补句式中**
   4. 介宾
3. 非谓语动词作表语
   1. 常见形式 ：系动词（一般为be动词）+to do（doing可能会有歧义）
   2. **主系表的结构中主语和表语都可以用非谓语动词但是要前后一致**

### 非谓语动词作定语，状语

1. 非谓语动词修饰名词，作定语
   1. **通常翻译为……的……**
   2. **doing 表示主动，done表示被动，to do 表示目的**
2. 非谓语动词修饰句字，做状语
   1. 非谓语动词做状语为了表示的更加明确，可以加上从属连词

# 长难句

## 并列句

### 并列句的构成

1. 构成：多个简单句+并列连词构成并列句
2. 表示顺接的并列连词
   1. and ，both and 表示和，什么和什么
   2. not only ……but also ， not only but ，not just ……but，not merely……but 表示不但而且
3. 表示转折的词
   1. yet，but ……但是……
4. 表示选择的连词
   1. or，either or ，表示……或者……
   2. neither or，表示既不……也不
5. 表示因果
   1. ……for…… 因为
   2. ……so…… 所以
6. **并列连词可以连接句子，词组，词等等**
7. **多个并列连词，前面的连词可以改成，号**

### 并列句的省略

1. 在并列句中相同的主语可以省略，其他不变
2. 相同成分可以省略，不单单是主语

## 主从复合句

### 名词性从句

#### 宾语从句

1. 写法
   1. 陈述句变为宾语从句，前加that ，其他不变
   2. 疑问句变宾语从句，将句式改为陈述句，中间用疑问句疑问词连接
   3. 一般疑问句变为宾语从句，该句式为陈述句，中间加whether，if连接句子
2. 位置
   1. 一般有三种，动宾，介宾，形容词宾
   2. 动宾：
      1. 及物动词+宾语从句
      2. 及物动词+宾语+宾语从句
   3. 介宾
      1. 介词+宾语从句
   4. 形容词宾
      1. 形容词+宾语从句
   5. **非谓语的及物动词也可以跟宾语从句**
3. 特殊：当宾语从句过长时，也可以用it做形式宾语，宾语从句后置

#### 表语从句

1. 写法和宾语从句相同，that不可以省略
2. 表语从句位于系动词之

#### 主语从句

1. 写法同宾语从句，that不可省略，只能用whether 表示是否
2. 可以位于句首或句尾，位于句尾需要有it作为形式主语

#### 同位语从句

1. 写法同宾语从句，常用that引导，不可省略
2. 同位语从句常在抽象名词之后
   1. 常见名词：idea，opinion，view，fact（事实），evidence（证据），question（问题，疑问），doubt(怀疑)，reason（原因），demand（要求），theory（理论），belief(信念)，possibility（可能性)，chance（机会，可能性），hope，contention（争论），guarantee（保证）

#### 名词性从句的要点

1. 分为四类
2. 写法，连接词+陈述句

### 定语从句

1. 写法：先行词+关系词+陈述语句
2. 关系词
   1. 人：who，whom（宾），that
   2. 事：which，that
   3. 人的物：whose
   4. 时间：when
   5. 地点：where
   6. 原因：why
   7. **定语从句每个关系词必须做成分**
3. 分类
   1. 限定性定语从句
   2. 非限定性定语从句
   3. 区别
      1. 有无逗号，有非限定，无限定
      2. 先行词范围是否明确，明确非限定，不限定
4. 补充
   1. 同位语从句和限定性从句的区别
      1. that是否做成分，是否可以省略
   2. 特殊的非限定从句
      1. which，as修饰整个句子
   3. 介词提前的定语从句
   4. 

### 状语从句

1. 分类：九类状语从句
2. 定义：一个句子做状语，表达描述性信息
3. 写法：从属连词+完整的陈述句
4. 状语从句的位置：主句前，主句后，主句中
5. 分类：
   1. **时间状语从句**
      1. when，while，as 当……时候
      2. before：在……之前
      3. after：在……之后
      4. since：自从
      5. until：直到
      6. as soon as ：一……就
      7. by the time ：截止到……
      8. each time、every time ：每一次
      9. the next time ：下一次
   2. 地点状语从句
      1. where
   3. **原因状语从句**
      1. because ：因为
      2. since：因为
      3. as：因为
      4. now that ：既然
   4. **结果状语从句**
      1. such that ：太……所以
      2. so that ：太……所以
      3. so that：所以
   5. 目的状语从句
      1. so that、in order that ：为了，目的是
   6. **条件状语从句**
      1. if 如果
      2. unless ：除非
      3. once：一旦
      4. as long as，so long as ：只要
   7. **让步状语从句**
      1. although，though：虽然
      2. even if，even though：即使
      3. while：虽然
      4. however，no matter how：不管怎样
   8. 比较状语从句
      1. than，as：比较
   9. 方式状语从句
      1. as ：按照……
      2. as if：似乎

## 特殊句式

### 虚拟

1. 虚拟表示不是真的，或者假设也不成真
2. 条件句可以分为：**真实条件句，虚拟条件句**
3. 写法
   1. 从句
      1. 过去：had done
      2. 现在：did ,were
      3. 将来：did，were，were to do，should do
   2. 主句
      1. would，could，should，might+
      2. 过去+have done
      3. 现在+do
      4. 将来+do
   3. 从句往前推，主句四加一，将来同现在
4. 名词性从句的虚拟
   1. 当主句有建议，命令，要求，名词性从句必须要用虚拟，谓语动词改为(should) do

### 倒装

1. 主谓倒装，谓语动词放在主语前
2. 分类：全部倒装，部分倒装
   1. 全部倒装：将谓语动词后面和前面互换
      1. there be句型
      2. there be+名词+介词短语
      3. there be+时态+情态
   2. 部分倒装：将陈述句改为疑问句
      1. 否定词位于句首
      2. only位于句首
      3. 虚拟语气if省略
      4. 例句：not only did they develop
         1. 正常：they developed

### 强调

1. 构成：itis，was加强调部分+that+剩余部分
2. 强调句不可用形容形容词和动词
3. 强调句要和主语从句后置区分
   1. 去it that 判断是否句子完整
   2. 强调句不可以修饰形容词

# 长难句

## 基本结构的长难句分析

### 断开长难句

#### 标点断开

1. 逗号
2. 冒号
3. 破折号
4. 分号

#### 连接词断开

1. 从句的开始于连接词
2. 结束语标点，下一个连接词，第二个谓语动词之前

#### 分析主谓

1. 主谓+主谓（从句）
2. 主主谓谓，中间为从句

### 简单化长难句

1. 简单句的核心成分：主谓宾表补
2. 非核心：形容词，副词，介词，同位语，插入语，非谓语动词
3. 取出核心成分-》添加每个成分的修饰内容

## 特殊结构的长难句的分析

### 分裂结构

1. 插入语
   1. 一般有逗号，破折号等等可以进行插入分裂句子
2. 从句造成的分裂
   1. 一般从句过长时，会造成从句后移使整个句子结构美观

### 嵌套结构

1. 一个从句还没有说完另外一个从句已经开始
2. 找到从句-》理清嵌套关系

### 平行结构

1. 平行结构的多个成分都是一致的
2. 名词词组的平行结构
3. 动词的平行并列
4. 介词的平行并列
5. 句子的平行并列

1. 解决分裂结构的关键是还原成连贯
2. 解决嵌套结构的关键是分层次
3. 解决平行结构的关键是沥青多个并列
